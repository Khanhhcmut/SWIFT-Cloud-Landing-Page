Application Preview

App Dashboard

  • • Overview of total stored files categorized by type such as documents, images, videos, audios and others.
  • • Visual storage distribution chart to quickly understand how storage space is being used.
  • • Real-time statistics showing storage consumption compared to the total available capacity.
  • • Quick navigation cards for different file categories allowing users to access data faster.
  • • Clean and minimal interface designed to give users a clear snapshot of their cloud storage.

My File Manager

  • • Folder navigation system for organizing files inside cloud storage.
  • • Upload files or entire folders directly to OpenStack Swift storage.
  • • Search functionality to quickly locate files or directories.
  • • Display of detailed file information including size and last updated timestamp.
  • • Simple interface similar to traditional file explorers for easier user interaction.
